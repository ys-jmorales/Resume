# Resume – John Philip Cesar P. Morales

HTML/CSS CV used as a **client endorsement profile** by You_Source Inc.

## Structure

```
index.html   – CV markup
styles.css   – Print-optimized styles (Letter size)
```

## Viewing

Open `index.html` in any browser. To export as PDF, use the browser's **Print** dialog and select **Save as PDF** with **Letter** paper size and the margins set to **None** (margins are defined in CSS via `@page`).

## Stack

- Plain HTML + CSS — no build tools or dependencies
- [Google Fonts – Roboto](https://fonts.google.com/specimen/Roboto)
- [Google Material Icons](https://fonts.google.com/icons)
